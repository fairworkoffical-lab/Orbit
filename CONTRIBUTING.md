# Contributing to Orbit Hospital OS

Thank you for your interest in contributing to Orbit! We welcome contributions from the community to help make this the best open-source hospital management system.

## ğŸ¤ How to Contribute

### 1. Fork the Repository
Click the "Fork" button on the top right of the repository page to create your own copy of the project.

### 2. Clone Your Fork
```bash
    git clone https://github.com/<your-username>/Orbit.git
    cd Orbit
```

### 3. Create a Branch
Create a new branch for your feature or bug fix:
```bash
git checkout -b feature/amazing-new-feature
# or
git checkout -b fix/bug-description
```

### 4. Make Changes
- Write clear, concise code.
- Follow the existing folder structure (`src/app`, `src/components`, etc.).
- Ensure your code compiles with `npm run build`.

### 5. Commit & Push
```bash
git add .
git commit -m "feat: Add amazing new feature"
git push origin feature/amazing-new-feature
```

### 6. Open a Pull Request
Go to the original repository and click "New Pull Request". Describe your changes and link any related issues.
